## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/yourusername/portfolio.git
```

### Navigate to project

```bash
cd portfolio
```

### Install dependencies

```bash
npm install
```

### Start development server

```bash
npm start
```

The application will run at:

```
http://localhost:5173
```
